Raspberry Pi Nexus Controller Setup

Hardware

  • Raspberry Pi 3/4
  • 4-Relay HAT
  • Wired to your Nexus valves/pumps

Installation

  sudo apt update
  sudo apt install python3-pip mosquitto
  pip3 install -r backend/requirements.txt
  cd backend
  uvicorn main:app --host 0.0.0.0 --port 8000
